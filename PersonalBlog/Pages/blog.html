Ayla Store – React JS

I spearheaded the development of Ayla Store, an E-Commerce mobile shop built using React JS with Redux integration. The platform offers seamless user authentication, utilizing a sign-in feature that verifies user information from a dummy JSON API. The home page showcases a curated selection of phone products retrieved dynamically from the API's phone category. Users can effortlessly search and explore products, with detailed product pages allowing for easy addition to the shopping cart. The shopping cart functionality is powered by Redux, ensuring smooth synchronization and management of user-selected items. A dedicated cart page provides users with a comprehensive overview of their selected items, with a streamlined checkout process facilitated by PayPal integration for secure payments. Additionally, the platform includes a sign-out button, enhancing user convenience and security. Ayla Store represents a sophisticated and user-centric E-Commerce solution, blending cutting-edge technology with intuitive design to deliver an unparalleled shopping experience.

ProgZone – Graduation Project – React JS & React Native

ProgZone is an all-inclusive online platform tailored for programming aficionados, catering to diverse user roles including admins, instructors, and students. Its intuitive interface facilitates seamless navigation, offering features like sign up, sign in, home, instructor profiles, course categories, and a comprehensive library of course videos. Powered by a robust database backend, it efficiently manages user data for admins, instructors, and students, with Axios handling seamless data fetching. Additionally, we create ProgZone for mobile using React Native, ensuring that users can access all these features on the go, providing a consistent and responsive experience across both web and mobile platforms.
